JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Delcare a variable using var.
  • Control flow is the order in which a computer executes code in a script.
  • It is best practice to write variables in camelCase in which the first word in a multi-word variable is lower case and the first letter in each word of all following words are capitalized.
  • Developers oftern use uppercase variable identifiers for immutable values (ones you don't want to change) but camelCase for mutable vaules (objects and arrays).
